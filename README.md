Stabilize UI Layout is a Codex skill for diagnosing and fixing frontend layout jitter, including scrollbar-induced page shifts, horizontal overflow, dynamic text width changes, and unstable filter or toolbar controls. It provides a practical workflow for identifying whether movement is global or component-level, applying layout-only fixes such as scrollbar-gutter: stable, overflow-x: hidden, fixed/minimum widths, shrink-0, and tabular-nums, and verifying the result without changing business logic, routing, filtering, or data behavior.
该skill可以让agent处理常见的页面元素布局抖动的问题，像是滚动条造成的布局抖动等等。
